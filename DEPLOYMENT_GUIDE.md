# **Deployment Guide: GitHub Pages**

Follow these steps to host **Ezra's Money Machine** for free on GitHub.

### **Step 1: Create the Repository**

1. Log in to [GitHub.com](https://github.com).  
2. Click the **\+** icon (top right) \-\> **New repository**.  
3. Repository name: ezra-money (or whatever you like).  
4. Select **Public**.  
5. Check the box **"Add a README file"**.  
6. Click **Create repository**.

### **Step 2: Upload Files**

1. Click **Add file** \-\> **Upload files**.  
2. Drag and drop the following files into the box:  
   * index.html  
   * taitt\_fin\_horizontal\_banner.png (Your logo file)  
   * LICENSE  
   * QUICKSTART.md  
3. In the "Commit changes" box at the bottom, type "Initial launch".  
4. Click **Commit changes**.

### **Step 3: Turn on the Website**

1. Click the **Settings** tab (gear icon) at the top of your repository.  
2. On the left sidebar, click **Pages** (under the "Code and automation" section).  
3. Under **Build and deployment** \> **Branch**:  
   * Select **main** from the dropdown menu.  
   * Ensure the folder is set to **/(root)**.  
   * Click **Save**.

### **Step 4: Share**

1. Wait about 1-2 minutes.  
2. Refresh the Settings \> Pages page.  
3. You will see a link at the top: Your site is live at...  
4. Copy that link and send it to Ezra\!
